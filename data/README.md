# Data Folder

Add your heart disease dataset here as `heart_disease.csv`.

Expected columns:
- `age`
- `sex`
- `cp`
- `trestbps`
- `chol`
- `fbs`
- `restecg`
- `thalach`
- `exang`
- `oldpeak`
- `target`

If you are using the UCI Heart Disease dataset, convert the target to 0/1 values and rename columns to match the expected schema.
